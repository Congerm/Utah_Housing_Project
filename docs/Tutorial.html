Tutorial: Accessing and Exploring the Data

This tutorial walks through fetching, cleaning, and doing initial exploration of the Utah housing affordability dataset. If you want to follow along interactively, download and run getting_started.ipynb — it contains all the same steps as this page in a ready-to-run Jupyter notebook.

1. Installation

Install the utah-housing package before running any code:

pip install utah-housing

You will also need a free Census API key. Obtain one at https://api.census.gov/data/key_signup.html, then set it as an environment variable:

import os
os.environ["CENSUS_API_KEY"] = "your_census_api_key_here"

Tip: Never commit your API key to git. Consider storing it in a .env file and loading it with python-dotenv.

2. Fetching the Data

The fetch_all_years function pulls ACS 5-year estimates for all Utah census tracts across the years you specify. This calls the Census API once per year, so it may take a minute or two.

import pandas as pd
from utah_housing import fetch_all_years, OUTCOME, PREDICTORS

df = fetch_all_years(years=range(2009, 2024))
print(f"Total rows: {len(df):,}")
df.head()

Once fetched, save the result to CSV so you don’t need to hit the API again:

import os
os.makedirs("data", exist_ok=True)
df.to_csv("data/utah_housing.csv", index=False)
print("Saved to data/utah_housing.csv")

On future sessions, reload from the CSV directly:

df = pd.read_csv("data/utah_housing.csv")

3. Initial Exploration

Get a quick sense of the dataset’s shape and the variable groupings provided by the package:

print("Shape:", df.shape)
print("\nOutcome variable:", OUTCOME)
print("Predictors:", PREDICTORS)

Check column types and look for missing values in the key analysis columns:

df.dtypes
analysis_cols = ["year", "GEOID"] + [OUTCOME] + PREDICTORS
df[analysis_cols].isnull().sum()

Get a statistical summary of the full dataset:

df.describe()

4. Cleaning the Data

The package returns fairly clean data, but you may want to drop rows missing the outcome variable and extract a readable county name for easier filtering:

df_clean = df.dropna(subset=[OUTCOME]).copy()

df_clean["county_name"] = df_clean["NAME"].str.extract(r",\s*(.+?)\s+County", expand=False)

print(f"Rows after cleaning: {len(df_clean):,}")
df_clean.head()

5. Subsetting by County or Year

Filter to a specific county or a specific year for focused analysis:

salt_lake = df_clean[df_clean["county_name"] == "Salt Lake"]
print(salt_lake.shape)

df_2023 = df_clean[df_clean["year"] == 2023]
print(df_2023.shape)

6. Basic Summary Statistics

summary_cols = [OUTCOME] + PREDICTORS
df_clean[summary_cols].describe().round(2)

The data is now ready for EDA and modeling. See the Technical Report for the full analysis, or run getting_started.ipynb to work through these steps yourself.
