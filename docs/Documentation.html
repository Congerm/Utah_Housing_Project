Documentation

Overview

With this project, we hoped to answer the question: what factors are driving an decrease in single-family-home availability in Utah? We used US Census data, available through census.gov to facilitate our analysis.

Data Source

The data comes from various tables available through the US Census Bureau. One observation corresponds to a single census tract in a single year, meaning there are hundreds of observations per year. We pull for years 2009-2023, but analyze only 2013-2023, that being the year economists have generally decided that the US economy recovered from the 2008 recession.

Table Details
B25024 Units in Structure
B25001 Housing Units
B25003 Tenure (Owner Occupied vs Renter Occupied)
B25008 Total Population in Occupied Housing Units by Tenure
B25119 Median Household Income in the Past 12 Months (in 2024 Inflation-Adjusted Dollars) by Tenure
B25032 Tenure by Units in Structure
B25088 Median Selected Monthly Owner Costs (Dollars) by Mortgage Status
B19013 Median Household Income in the Past 12 Months (in 2024 Inflation-Adjusted Dollars)

Key Variables

Note that three of the five predictors are derived from other variables; while it seems counterintuitive to pull eight tables and only use five predictors, each table is necessary for the analysis.

Outcome Variable

Variable Description
median_owner_costs_with_mortgage Median monthly owner costs for households with a mortgage

Predictors

Variable Description
pct_sf_renter_occupied Fraction of single-family homes that are renter-occupied (renter_single_family / single_family_units)
median_household_income Median household income in the past 12 months (inflation-adjusted)
owner_renter_income_gap Difference in median income between owner- and renter-occupied households (median_income_owner_occupied - median_income_renter_occupied)
pct_vacant Fraction of housing units that are vacant (vacant_units / total_units_b25002)
pop_in_occupied_total Total population living in occupied housing units

Python Package: utah-housing

The utah-housing package provides functions to fetch ACS data directly from the Census API and compute derived housing affordability metrics.

Installation

pip install utah-housing

Note: A free Census API key is required. Obtain one at https://api.census.gov/data/key_signup.html and set it as an environment variable:

export CENSUS_API_KEY="your_key_here"

Core Functions

from utah_housing import fetch_year, fetch_all_years, OUTCOME, COMPLEX_PREDICTORS as PREDICTORS

# Fetch ACS 5-year estimates for all years
df = fetch_all_years(years=range(2009, 2024))

# Inspect variable groupings
print("Outcome:", OUTCOME)
print("Predictors:", PREDICTORS)
Function / Constant Description
fetch_year(year) Fetches ACS 5-year estimates for all Utah census tracts for a single year
fetch_all_years(years) Fetches ACS estimates for all Utah census tracts across a range of years
OUTCOME String constant: "median_owner_costs_with_mortgage"
PREDICTORS List of all five predictor variable names used in the model

## **Overview**

With this project, we hoped to answer the question: what factors are driving an decrease in single-family-home availability in Utah? We used US Census data, available through [census.gov](https://census.gov) to facilitate our analysis. 

---

## **Data Source**

The data comes from various tables available through the US Census Bureau. One observation corresponds to a single census tract in a single year, meaning there are hundreds of observations per year. We pull for years 2009-2023, but analyze only 2013-2023, that being the year economists have generally decided that the US economy recovered from the 2008 recession.  

| Table | Details |
|----------|---------|
| **B25024** | Units in Structure |
| **B25001** | Housing Units |
| **B25003** | Tenure (Owner Occupied vs Renter Occupied) |
| **B25008** | Total Population in Occupied Housing Units by Tenure |
| **B25119** | Median Household Income in the Past 12 Months (in 2024 Inflation-Adjusted Dollars) by Tenure |
| **B25032** | Tenure by Units in Structure |
| **B25088** | Median Selected Monthly Owner Costs (Dollars) by Mortgage Status |
| **B19013** | Median Household Income in the Past 12 Months (in 2024 Inflation-Adjusted Dollars) |

## **Key Variables**

Note that three of the five predictors are derived from other variables; while it seems counterintuitive to pull eight tables and only use five predictors, each table is necessary for the analysis. 

### Outcome Variable

| Variable | Description |
|----------|-------------|
| `median_owner_costs_with_mortgage` | Median monthly owner costs for households with a mortgage |

### Predictors

| Variable | Description |
|----------|-------------|
| `pct_sf_renter_occupied` | Fraction of single-family homes that are renter-occupied (`renter_single_family / single_family_units`) |
| `median_household_income` | Median household income in the past 12 months (inflation-adjusted) |
| `owner_renter_income_gap` | Difference in median income between owner- and renter-occupied households (`median_income_owner_occupied - median_income_renter_occupied`) |
| `pct_vacant` | Fraction of housing units that are vacant (`vacant_units / total_units_b25002`) |
| `pop_in_occupied_total` | Total population living in occupied housing units |

## **Python Package: `utah-housing`**


The `utah-housing` package provides functions to fetch ACS data directly from the Census API and
compute derived housing affordability metrics.

### Installation

```bash
pip install utah-housing
```

> **Note:** A free Census API key is required. Obtain one at
> <https://api.census.gov/data/key_signup.html> and set it as an environment variable:
>
> ```bash
> export CENSUS_API_KEY="your_key_here"
> ```

### Core Functions

```{python}
#| eval: false
from utah_housing import fetch_year, fetch_all_years, OUTCOME, COMPLEX_PREDICTORS as PREDICTORS

# Fetch ACS 5-year estimates for all years
df = fetch_all_years(years=range(2009, 2024))

# Inspect variable groupings
print("Outcome:", OUTCOME)
print("Predictors:", PREDICTORS)
```

| Function / Constant | Description |
|---------------------|--------------|
| `fetch_year(year)` | Fetches ACS 5-year estimates for all Utah census tracts for a single year |
| `fetch_all_years(years)` | Fetches ACS estimates for all Utah census tracts across a range of years |
| `OUTCOME` | String constant: `"median_owner_costs_with_mortgage"` |
| `PREDICTORS` | List of all five predictor variable names used in the model |
